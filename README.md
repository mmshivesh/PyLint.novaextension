# PyLint

This extension adds support for PyLint in Nova.

## Requirements

Needs PyLint to be installed:

```bash
pip install pylint
```

(or) 

```bash
pip3 install pylint
```

## Configuration

Default PyLint path is `/usr/local/bin/pylint`. Configure how Nova highlights issues raised by PyLint.
